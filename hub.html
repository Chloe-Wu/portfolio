Problem

How might we make students living in an apartment interact and spend more time with each other while having a busy schedule?



Solution

    Based on the design requirements we came up with the following design solutions.
  • Colored lights as an abstract way of representing the availability and mood of the roommates.
  • Place the product in the middle of the living room, on the floor, to be easily visible from anywhere in the living room.
  • The object must be made attractive so that it always attracts the attention of the roommates.
  • Give reasons to make roommates come more often to the living room.

Test



Size test

Based on the size limit of 3D printer, we decide to print our model piece by piece (6 sides, 2 bottoms, and two lids). And we did the size testing to make sure the model size can be printed and fitted.


Acrylic board test

In order to reduce the vision hurt, we engrave acrylic board to make some translucent effect.


Light test

We test the LED strip light to make sure the user can control the color and brightness. And we test the strip length in each side and the method of arranging wires.
